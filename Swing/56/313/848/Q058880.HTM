A presença de um elemento atômico em um gás pode ser determinada verificando-se as energias dos fótons que são emitidos pelo gás, quando este é aquecido.

No modelo de Bohr para o átomo de hidrogênio, as energias dos dois níveis de menor energia são:

 

E1 = - 13,6 eV

E2 = - 3,40 eV.

 

Considerando-se essas informações, um valor possível para a energia dos fótons emitidos pelo hidrogênio aquecido é

a)   - 17,0 eV.

b)   - 3,40 eV.

c)   8,50 eV.

d)   10,2 eV.
